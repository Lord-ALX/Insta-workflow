PROJET INSTA WORKFLOW - RAILWAY + WIDGET HTML

Contenu:
- server.js : backend Railway
- package.json : dependances Node
- .env.example : variables d'environnement
- widget_insta_workflow.html : front complet HTML+CSS+JS en un seul fichier

Variables Railway:
- CORS_ORIGIN=dental3d.mc
- API_TOKEN=votre_token
- GEMINI_API_KEY=votre_cle_gemini

Widget HTML:
Modifier dans widget_insta_workflow.html:
const CONFIG = {
  API_BASE_URL: "https://VOTRE-APP.up.railway.app",
  API_TOKEN: "VOTRE_TOKEN_API"
};

Commandes:
npm install
npm start
